# PDF Editor

A simple web-based PDF editor built with React, TypeScript, and Konva.js.  
Users can import PDFs, draw/edit on top of them, and export the modified result.

---

## Features

- Import PDF files  
- Create new blank PDFs  
- Add and edit text  
- Freehand drawing  
- Add shapes (square, circle, triangle)  
- Image upload  
- Eraser  
- Add new pages  
- Export edited PDF  

---

## Tech Stack

- **React (TypeScript)**
- **Konva.js** → canvas editing (drawing, shapes, text)
- **react-pdf-viewer** → PDF rendering
- **pdf-lib** → generating/exporting PDFs
- **Tailwind CSS** → UI styling
